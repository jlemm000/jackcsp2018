Go Fish

This python program is a simulation of the card game Go Fish. Click to download.

Abstraction is used throughout this project, as many functions are called from within one another when necessary in order to efficiently run the necessary parts of the program. A package, random, was also imported, which contains pre-made abstractions that I used to randomly deal cards.
